## 使用文档

### 简介

该脚本把目录下所有 `.md` 文件已 hexo 或 gitbook 等方式展示出来，方便本地浏览。

### 依赖环境

* python2.7
* node.js
* hexo
* python模块
    * shutil
    
### 如何使用

下载脚本，执行命令 

方式1：

`$ python server.py gitbook iOSAdvanced 2`

gitbook: 将要生成的博客文件夹          
iOSAdvanced:  整理好的文章

方式2：

`$ python server.py `

请输入博客名字:                   
请输入文章目录:                
请输入服务类型序号:                 
1 : hexo;               
2 : gitbook;
